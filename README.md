# InterviewQA
面试题app：

app使用 Kotlin+MVP+RxJava2+Retrofit2；（开源源码为1.0.2版本）

功能：  
1、面试题列表  
2、面试题详情  
3、添加面试题  
4、根据标题字段模糊搜索  
5、问题反馈  


App下载地址：

1.0.2版本：https://www.pgyer.com/android_iqa


后续计划：
- 新增用户注册、登录
- 全新的编辑模式-MarkDown
- 多种面试题类型：Android，Java，C++，计算机网络，算法/机器学习，前端等等.
- 收藏功能、本地缓存功能
- 添加专项练习题、查看用户资料
- 发布面试心得、用户点赞、评论
- 面试时间预约提醒
- 积分兑换功能
