# InterviewQA
Android面试题app：
app使用 Kotlin+MVP+RxJava2+Retrofit2；
功能：  
1、面试题列表  
2、面试题详情  
3、添加面试题  
4、根据标题字段模糊搜索  
5、问题反馈  



启动页不合主题：因为不会PS作图，就只好在以前项目里找了张启动图，见谅啊~~~
