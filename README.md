# InterviewQA
Android面试题app：
app使用 Kotlin+MVP+RxJava2+Retrofit2；
功能：\n
1、面试题列表\n
2、面试题详情\n
3、添加面试题\n
4、根据标题字段模糊搜索\n
5、问题反馈\n

启动页不合主题：因为不会PS作图，就只好在以前项目里找了张启动图，见谅啊~~~
